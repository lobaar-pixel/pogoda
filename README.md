Описание программы
Weather Diary — это простое GUI-приложение для ведения дневника погоды.
Программа позволяет:
Добавлять записи о погоде (дата, температура, описание, наличие осадков);
Просматривать список всех записей;
Фильтровать записи по дате или температуре;
Сохранять данные в файл формата JSON;
Загружать данные из JSON-файла.
🚀 Как запустить программу
Установите Python 3.10+ на ваш компьютер.
Установите необходимые библиотеки:
pip install -r requirements.txt
Запустите файл:
python main.py
🛠 Как пользоваться
Введите дату, температуру, описание погоды и отметьте наличие осадков.
Нажмите кнопку "Добавить запись".
Для фильтрации используйте соответствующие поля и кнопки.
Для сохранения данных нажмите "Сохранить в JSON".
Для загрузки данных из файла используйте "Загрузить из JSON".
📂 Пример использования
Добавление записи:
Дата: 2024-05-01
Температура: +15°C
Описание: Солнечно
Осадки: Нет

Фильтрация по температуре:
Показать все записи с температурой выше +10°C.
